表格布局实现居中布局

其实 CSS 布局流行起来之前一直使用的是表格布局。在居中布局这种场景下，表格布局也很适用。 此布局的关键点在于：
  1. 父元素上 3 个样式设置：display: table-cell; text-align: center; vertical-align: center;；
  2. 子元素上设置 display: inline-block;。
display: table-cell; 会使元素表现得像表格的单元格，而 text-align: center; 和 vertical-align: center; 会使元素水平和垂直居中。 display: inline-block; 会使元素表现得像行内块元素，可以让元素水平排列。
