CSS 盒模型

1：CSS 盒模型的概念

CSS 盒模型是指浏览器将每个 HTML 元素都看作一个矩形盒子，并且这个盒子由四个部分组成：内容区（Content）、内边距（Padding）、边框（Border）和外边距（Margin）。这些部分共同决定了元素在页面上的布局和显示方式。

有两种盒子模型

  • 标准盒模型（W3C 盒模型）：

    在标准盒模型中，元素的宽度和高度只包括内容区的尺寸，不包括内边距、边框和外边距。

  • IE 盒模型（IE Box Model）：

    在 IE 盒模型中，元素的宽度和高度包括内容区、内边距和边框，但不包括外边距。

2：CSS 盒模型的注意点

  1. Padding 不能为负值，而 Margin 可以为负值
  2. 背景色会平铺到非 Margin 的区域
  3. Margin-top 传递的现象
  4. Margin 上下叠加的现象

3：IE 盒模型的使用场景

IE 盒模型在早期的浏览器中被广泛使用，但现代浏览器都遵循标准盒模型。一般，为了兼容性，建议使用标准盒模型。 如果需要使用 IE 盒模型，可以通过设置 box-sizing: border-box; 来实现。但是有些情况下使用 IE 盒模型可能会更方便，例如在处理一些复杂的布局时，可以更容易地控制元素的尺寸和间距。类比 Android 中的 View，它的盒子模型与 IE 盒模型是一致的。
