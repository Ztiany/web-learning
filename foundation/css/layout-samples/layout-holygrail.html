圣杯布局简介

圣杯布局和双飞翼布局解决的问题是一样的，就是两边定宽，中间自适应的三栏布局，中间栏要放在文档流前面以优先渲染。 圣杯布局的来历是 2006 年发在 a list part 上的这篇文章：In Search of the Holy Grail.

圣杯布局的实现：

  1. 三者都设置向左浮动。
  2. 设置中间的宽度为 100％。
  3. 设置负边距 margin-left。左部分设置 -100%，右部分设置 -width px。
  4. 设置容器的 padding 值给左右两部分留出空间。
  5. 设置左右部分为相对定位。左部分 left 的为 -width，右部分 right 为 -width。

注意：圣杯布局有个问题，当面板的中间部分比两边的子面板宽度小的时候，布局就会乱掉。
