Display 属性

Display 属性说明

Display 属性用于指定元素的显示类型。它可以控制元素在页面上的布局方式和行为。 常见的 display 属性值包括： block、inline、inline-block、none、table、 flex、grid等。它们的作用如下：

  • block：元素会被呈现为块级元素，占据一整行。
  • inline：元素会被呈现为行内元素，不会占据一整行。
  • inline-block：元素会被呈现为行内块级元素，既可以设置宽高，又可以在同一行内显示。
  • none：元素不会被渲染，也不会占据空间。会禁用 CSS 动画/过渡。注意与 visibility:hidden 的区别。
  • table：元素会被呈现为表格布局，类似于 HTML 中的 table 元素。
  • flex：元素会被呈现为弹性盒子布局，子元素成为弹性项目，可以灵活地调整子元素的大小和位置。
  • grid：元素会被呈现为网格布局，可以创建复杂的二维布局，子元素放置于网格单元格。

另外还需要注意：

  • 浮动/定位可能隐式改变 display 值（如浮动元素变为 block）
  • display 属性的值可以继承，但通常不会被子元素继承。
  • display 属性的值可以通过 JavaScript 动态修改，以实现动态布局效果。

display:none 与 visibility:hidden 的区别

  1. “display:none”的元素被隐藏之后，不占据原来的位置。也就是说彻底地消失了，看不见也摸不着。
  2. “visibility:hidden”的元素被隐藏之后，依然占据原来的位置。也就是说并没有彻底消失，看不见但摸得着。

display:table, table-cell, table-row 的作用

display: table 系列属性允许非表格元素模拟 HTML 表格行为，提供灵活的布局能力。
  • display: table：模拟表格容器，类似于 HTML 中的 table 元素。
  • display: table-row：模拟表格行，类似于 HTML 中的 tr 元素。
  • display: table-cell：模拟表格单元格，类似于 HTML 中的 td 或 th 元素。
  • display: table-caption：模拟表格标题，类似于 HTML 中的 caption 元素。
  • display: table-column：模拟表格列，类似于 HTML 中的 col 元素。
  • display: table-column-group：模拟表格列组，类似于 HTML 中的 colgroup 元素。
  • display: table-header-group：模拟表格头部组，类似于 HTML 中的 thead 元素。
  • display: table-footer-group：模拟表格底部组，类似于 HTML 中的 tfoot 元素。
  • display: table-row-group：模拟表格行组，类似于 HTML 中的 tbody 元素。

这些属性可以用于创建复杂的布局，而不需要使用实际的 HTML 表格元素。其中主要关注 table、table-cell 和 table-row 这三个值，

  1. table：
    • 元素被呈现为块级表格，类似于 table 元素。
    • 元素的直接子元素会被解释为表格行（tr）。
    • 表格行内部的直接子元素会被解释为表格单元格（td 或 th）。
  2. table-cell：
    • 元素被呈现为一个表格单元格，类似于 td 或 th 元素。
    • 元素会以表格单元格的方式对齐和布局。
    • 元素默认会填充可用空间，如果需要，可以使用 width 和 height 属性进行调整。
  3. table-row：
    • 元素被呈现为一个表格行，类似于 tr 元素。
    • 元素内部的直接子元素会被解释为表格单元格（td 或 th）。
    • 表格行内的元素会自动扩展以填充可用空间。

table-cell 应用：图片垂直居中于元素

在 CSS 中，display:table-cell 可以让元素以表格单元格的形式呈现。也就是说，table-cell 类型的元素具备 td 元素的特点。 而 table-cell 类型的元素可以通过设置 vertical-align: middle 来实现垂直居中。

table-cell 应用：等高布局

使用 table-cell 可以轻松实现等高布局。通过将父元素设置为 display: table， 子元素设置为 display: table-cell，可以使所有子元素的高度相等。 这是因为 table-cell 元素会自动调整高度以适应内容。

table-cell 应用：自动平均划分元素

如果我们想要使用 ul 来实现下面这种布局，一般都会用 float 来实现，并且还得精准计算每一个li的宽度。 但是如果给每个 li 元素都定义 “display:table-cell”，我们就不用显得如此麻烦。而且会自动平均划分元素。

去除 inline-block 元素间距

inline-block 元素之间是有间距的。在实际开发中，这种间距有时会对我们的布局产生影响。 大多数时候为了不影响布局，我们需要去除 inline-block 元素的间距。在 CSS 中，我们可以使用 “font-size:0” 来去除 inline-block 元素的间距。
