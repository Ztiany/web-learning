CSS 文件命名：

  1. reset.css：重置样式，重置元素的默认样式
  2. global.css：全局样式，全站公用，定义页面的基础央视
  3. themes.css：主题样式，用于实现换肤功能
  4. module.css：模块样式，用于模块的样式
  5. master.css：母版样式，用于母版页面
  6. columns.css：专栏样式
  7. forms.css：表单样式
  8. mend.css：补丁样式，用于维护、修改样式
  9. print.css：打印样式

为什么要用 CSS reset

现在浏览器有很多，主流的有 Chrome、Firefox、IE、Safari、Opera 等。不同的浏览器，默认样式也是不同的。 举个例子，ul 元素有缩进样式。在 Firefox 浏览器中，ul 元素的缩进是通过 padding 实现的；而在 IE 浏览器中，ul 元素的缩进是通过 margin 来实现的。 再比如 button 元素，在 IE、chrome、Firefox 等浏览器中的默认样式也是不同的。 推荐 CSS 专家 Eric Meyer 的重置样式表，这是国内外流行的重置样式表。 具体可参考：CSS Tools: Reset CSS。
