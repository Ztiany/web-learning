Display 属性

Display 属性说明

display:none 与 visibility:hidden 的区别

  1. “display:none”的元素被隐藏之后，不占据原来的位置。也就是说彻底地消失了，看不见也摸不着。
  2. “visibility:hidden”的元素被隐藏之后，依然占据原来的位置。也就是说并没有彻底消失，看不见但摸得着。

display:table, table-cell, table-row 的作用

display 属性的 table、table-cell 和 table-row 值可以用于模拟表格布局，具有以下特点：
  1. table：
    1. 元素被呈现为块级表格，类似于 table 元素。
    2. 元素的直接子元素会被解释为表格行（tr）。
    3. 表格行内部的直接子元素会被解释为表格单元格（td 或 th）。
  2. table-cell：
    1. 元素被呈现为一个表格单元格，类似于 td 或 th 元素。
    2. 元素会以表格单元格的方式对齐和布局。
    3. 元素默认会填充可用空间，如果需要，可以使用 width 和 height 属性进行调整。
  3. table-row：
    1. 元素被呈现为一个表格行，类似于 tr 元素。
    2. 元素内部的直接子元素会被解释为表格单元格（td 或 th）。
    3. 表格行内的元素会自动扩展以填充可用空间。

table-cell 应用：图片垂直居中于元素

在 CSS 中，“display:table-cell” 可以让元素以表格单元格的形式呈现。也就是说，table-cell 类型的元素具备 td 元素的特点。

table-cell 应用：等高布局

《ONE PIECE》（海贼王、航海王）简称"OP"，是日本漫画家尾田荣一郎 作画的少年漫画作品。在《周刊少年Jump》1997年34号开始连载。描写了拥有橡皮身体戴草帽的青年 路飞，以成为"海贼王"为目标和同伴在大海展开冒险的故事。

table-cell 应用：自动平均划分元素

如果我们想要使用 ul 来实现下面这种布局，一般都会用 float 来实现，并且还得精准计算每一个li的宽度。 但是如果给每个 li 元素都定义 “display:table-cell”，我们就不用显得如此麻烦。而且会自动平均划分元素。

当父元素定义 “display:table” 而子元素定义 “display:table-cell” 时，如果给父元素一定的宽度，父元素宽度就会根据子元素的个数进行自动平均划分。

去除 inline-block 元素间距

inline-block 元素之间是有间距的。在实际开发中，这种间距有时会对我们的布局产生影响。 大多数时候为了不影响布局，我们需要去除 inline-block 元素的间距。在 CSS 中，我们可以使用 “font-size:0” 来去除 inline-block 元素的间距。
